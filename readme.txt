Le WEB pertenece a una Muebleria. 

Tiene Sillas, Mesas y Sillones. 
Cada uno de los formualarios se encuentra en la barra de navegacion 

Se pueden cargar todos los productos, y consultar mediante cada formulario.

Existe a su vez una carga de usuarios, los cuales tambien pueden ser consultados mediante los formularios.

El modo de proceder para el testeo de la Web inicia validando la creacion de usuarios. 
Una vez tenga creado su usuario podra ingresar a la pagina en virtud de poder testear todas sus funcionalidades.
